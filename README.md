# Easy Route (Expo + pnpm)

A feature-organized Expo Router recreation of the provided accessibility map design.

## Run

```bash
pnpm install
pnpm start
```

## Structure

- `src/app` Ôćĺ Expo Router entry and navigation
- `src/components` Ôćĺ shared UI pieces
- `src/constants` Ôćĺ colors, theme, mock data
- `src/features/*/views` Ôćĺ screen-level UI
- `src/features/*/styles.ts` Ôćĺ feature-local styles
- `src/hooks`, `src/lib`, `src/providers`, `src/services`, `src/stores` Ôćĺ reserved for scale-up
